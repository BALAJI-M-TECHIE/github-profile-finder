.# GitHub Profile Finder

GitHub Profile Finder is a modern web application that allows users to search and explore GitHub profiles in real time using the GitHub REST API. The application displays detailed profile information, repository statistics, recent repositories, and search history through an intuitive and responsive user interface.

## Features

* 🔍 Search GitHub users by username
* 👤 View profile avatar, bio, company, location, and join date
* 📊 Display followers, following, repositories, and gists
* 📂 Fetch and display latest repositories
* ⭐ View repository stars and fork counts
* 📋 Copy GitHub profile URL with one click
* 🌙 Dark / Light theme toggle
* 💾 Search history using Local Storage
* ⚡ Real-time data fetching with GitHub REST API
* 📱 Fully responsive design for mobile, tablet, and desktop
* 🎨 Modern glassmorphism-inspired UI
* 🚨 Error handling for invalid usernames

## GitHub API

This project uses the GitHub REST API to retrieve user profile information and repository data.

API Endpoint Examples:

```text
https://api.github.com/users/{username}
https://api.github.com/users/{username}/repos
```
